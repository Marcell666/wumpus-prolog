Hunt the Wumpus *in Prolog*
---------------

Artificial intelligence agent for the Hunt the Wumpus game implemented in Prolog.

#### Running:

Run in shell throug `swipl`:

```sh
$ ./wumpus.sh src/naive.pl
```

Using SWI Prolog interface:

```shell
$ swipl
?- [src/main].
true.
?- [src/naive].
true.
?- run.
```
